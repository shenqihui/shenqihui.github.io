# 交流会是什么

* 有话题引导  {:&.moveIn}
* 你说一句我说一句
* 取长补短
* 要表述的时候，打断还是不打断另外再说
  * 只是，千万不要像老罗 VS 王自如一样，那是不礼貌的
  * 等我把话说完
* 怎么进行交流（普通谈话、通知、讨论，都算交流）：
  * 每次交流，都应有回应方 {:&.moveIn}
  * 都需要指定回应方，可能是一人、多人
