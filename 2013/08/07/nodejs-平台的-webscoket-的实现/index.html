    运行的方法为：

    1、安装node ，我的版本是 0.10.5，其他版本没进行测试；

    2、安装 websocket.io，命令如下 npm i -g websocket.io；

    3、环境弄好了，可以跑了，打开命令行，cd 到当前目录，跑 node server.js。

    4、直接用 chrome 以本地文件方式直接打开 index.html ，里面操作简单。其他浏览器没进行测试。当然以 http 形式访问更好。
